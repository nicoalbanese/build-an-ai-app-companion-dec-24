From here, you have several paths to explore:

- To learn more about the AI SDK, read through the [ documentation ](https://sdk.vercel.ai/docs).
- If you're interested in diving deeper with guides, check out the [ RAG (retrieval-augmented generation) ](https://sdk.vercel.ai/docs/guides/rag-chatbot) and [ multi-modal chatbot guides ](https://sdk.vercel.ai/docs/guides/multi-modal-chatbot).
- To jumpstart your first AI project, explore [ available templates ](https://vercel.com/templates?type=ai).

## Considerations
- Be careful about leaking API keys on GitHub
- Be sure to have authentication and rate limiting set up for your apps
